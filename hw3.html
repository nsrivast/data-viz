Homework #3 - Due July 7th

Download the source code for the "Hello, Chart" web page we explored in class. Make sure you also download the required Javascript libraries (jquery.min.js and highcharts.js) are included and place them in the correct directory relative to the html file.

Open the web page in your favorite browser (Chrome recommended) and ensure you are able to get the chart to load.

Make the following edits, one at a time:

  • change chart:type from "line" to "bar"
  • change chart:type from "line" to "column"
  • change chart:type from "line" to "scatter"

How does the graph change?

Go back to the "line" type. Search through the Highcharts Documentation for the way to change the scale of the y-axis from linear to logarithmic. (Hint: look under yAxis.)

If you get stuck, look through the demo charts on the Highcharts home page or search StackOverflow.
